# Install
(install requirements)

```
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
```

# Create Schedule
The input files provided is found in `data/case1` directory. Run the following command to ger the schedule.
```
 python ./main.py
```

 pytest
